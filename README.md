# i3-wallpaper-engine
A Linux alternative to the Windows software **Wallpaper Engine**,
## Features
- works with PolyBar
- multi-monitor support
- works with Picom

## Install
### Requirements
- xwinwrap
- mpv
- xrandr
- git
- python 3^
- npm

```
git clone https://github.com/9jh1/i3-wallpaper-engine
cd linux-wallpaper-engine
./install.sh
```
